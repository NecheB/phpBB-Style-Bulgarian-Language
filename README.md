## 🇧🇬 Български

### Име
**Bulgarian Language Pack**

### Автор
**NecheB © phpBB Bulgaria**

### Описание
Официален български езиков пакет за phpBB 3.3.x.

Пакетът съдържа преведени файлове на български език и е предназначен за всички форуми, използващи phpBB.

### Съвместимост
- ✅ phpBB 3.3.x
- ✅ UTF-8
- ✅ Български език

### Инсталация

1. Качете папката **bg** в:

```
language/
```

2. Влезте в **Administration Control Panel (ACP)**.

3. Отидете на:

```
Customise → Language Packs
```

4. Инсталирайте **Bulgarian** езиковия пакет.
